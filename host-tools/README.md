# host-tools

This directory is reserved for fixed host-side benchmark infrastructure.

Recommended scripts:

- `build.sh`
- `deploy_tftp.sh`
- `deploy_sd.sh`
- `capture_serial.sh`
- `interrupt_uboot.expect`
- `run_uboot_script.expect`
- `collect_logs.sh`

These scripts should be shared across all agents to keep the benchmark fair.
